# Asseria Gradnja - Modern Static Site (Tailwind)

Ovo je moderna statička verzija web stranice Asseria Gradnja (bez admin panela i backend integracije).

## Upute
1. Zamijeni slike (`naslovna.jpg`, `rad1.jpg`, `rad2.jpg`, `rad3.jpg`, `rad4.jpg`, `logo.png`) sa svojim slikama. 
2. Otvori `index.html` lokalno da testiraš ili uploadaj u GitHub repozitorij i aktiviraj GitHub Pages (Settings → Pages → main branch → /root).
3. Kontakt forma je demo (ne šalje mail). Za stvarno slanje integriraj email servis (Formspree, EmailJS ili backend).

## Datoteke u paketu
- index.html
- manifest.json
- service-worker.js
- README.md
- CNAME (ako želiš custom domenu)
